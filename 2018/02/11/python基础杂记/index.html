一.迭代器和生成器笔记总结

1.所有可以for循环的对象都是可迭代对象
2.生成器是一个迭代器，迭代器却不一定是生成器
3.迭代器是执行函数到yield处，并返回当前yield后变量值，下次执行时是从上一次的yield后出继续开始执行，直到再次执行到yield
4.迭代器与生成器都用func.next来取值，但是生成器是返回yield的值
5.迭代器相比列表可以减少内存的占用，一次只加载迭代中的一个
6.生成器可用来控制代码的执行
